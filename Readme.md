# Personal Portfolio Website

## Project Overview

This project is a responsive personal portfolio website developed using HTML, CSS, and Basic JavaScript. It showcases my personal profile, educational background, skills, projects, and contact information. The website is designed to be simple, responsive, and user-friendly.

## Features Implemented

* Responsive website design
* Home section
* About Me section
* Skills section
* Projects section
* Contact section
* Basic JavaScript interaction
* Mobile-friendly layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* GitHub Pages

## How to Run Locally

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
